Usage Examples

Auto-initialization with Data Attributes

<!-- Scrollable container (Y-axis only) --> <div data-scrollable="y"> <div>Your scrollable content here</div> </div> <!-- Scrollable container (both axes) --> <div data-scrollable="true"> <div>Your scrollable content here</div> </div> <!-- Zoomable container (with panning for better zoom origin) --> <div data-scrollable="true" data-zoomable="true"> <div>Your zoomable content here</div> </div> <!-- Zoomable with custom zoom range --> <div data-scrollable="true" data-zoomable="0.5-3"> <div>Content with zoom from 0.5x to 3x</div> </div>

Manual Initialization

// Import EasyScroller import { EasyScroller } from 'scroller'; // Get the container element (the one that should be scrollable) const container = document.getElementById('my-container'); // Initialize with options const easyScroller = new EasyScroller(container, { scrollingX: true, scrollingY: true, zooming: true, minZoom: 0.5, maxZoom: 3 }); // Use the API easyScroller.scrollTo(100, 200); easyScroller.zoomTo(1.5); const values = easyScroller.getValues(); // { left, top, zoom }

Data Attribute Options

data-scrollable values:
  • "true" - Enable scrolling on both X and Y axes
  • "x" - Enable scrolling on X-axis only
  • "y" - Enable scrolling on Y-axis only
data-zoomable values:
  • "true" - Enable zooming with default range (0.5x to 3x)
  • "min-max" - Enable zooming with custom range (e.g., "0.5-2" for 0.5x to 2x)
üí° Pro Tip: Combine data-scrollable="true" with data-zoomable for the best user experience. This enables proper zoom-from-cursor behavior and allows panning when zoomed in.
