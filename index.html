Zynga Scroller by zynga

Accelerated panning and zooming for HTML and Canvas

Dependencies

No dependencies.

Install

Just copy over the 3 JavaScript files from the "src" folder to your local JavaScript project and use include them in the order "Raf.js", "Animate.js" and "Scroller.js". It also supports Jasy - our build system - to auto-detect dependencies. In this case you need to download the "core" project (https://github.com/wpbasti/core) as well. Animate.js and Raf.js are then used from there.
